## ✅ Getting Started

```
# Clone this repository
git clone https://github.com/trinwin/storybook-next-ts-template

# Go into the repository
cd storybook-next-ts-template

# Install client dependencies
yarn install

# Start client on localhost:3000
yarn run dev

# Start Storybook
yarn storybook

# Run your test
yarn test
```
